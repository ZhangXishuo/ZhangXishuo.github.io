最近学习了Web中数据传输的一种新格式————Json，让我们一起学习一下神奇的Json吧！

概述

初识

这并不是第一次接触Json，其实之前在学习微信小程序开发文档时就用到了Json。

小程序有一个app.json的文件，用于配置小程序的各种属性，所以最初错误地以为Json就是一种配置文件格式。直到今天再次遇到，我们一起来探究一下Json到底是什么？

{
    "pages": [
        "pages/index/index",
        "pages/logs/index"
    ],
    "window": {
        "navigationBarTitleText": "Demo"
    },
    "tabBar": {
        "list": [{
            "pagePath": "pages/index/index",
            "text": "首页"
        },{
            "pagePath": "pages/logs/logs",
            "text": "日志"
        }]
    },
    "networkTimeout": {
        "request": 10000,
        "downloadFile": 10000
    },
    "debug": true
}

专业定义

我们先看看Json是什么，我们先看看维基百科的解释。

JSON（JavaScript Object Notation）是一种由道格拉斯·克罗克福特构想设计、轻量级的数据交换语言，以文字为基础，且易于让人阅读。尽管JSON是Javascript的一个子集，但JSON是独立于语言的文本格式，并且采用了类似于C语言家族的一些习惯。

JSON 数据格式与语言无关，脱胎于JavaScript，但目前很多编程语言都支持 JSON 格式数据的生成和解析。JSON 的官方 MIME 类型是 application/json，文件扩展名是 .json。

简单来说，Json就是一种起源于JavaScript且支持多种编程语言使用的一种数据交换语言。

语法

数据

"name" : "zhangxishuo"

左侧是名称，右侧是值。

Json的值可以是：数字、字符串、布尔类型、数组、对象、null。

对象

包含在一个花括号内，就是一个对象。

{
    "name"     : "zhangxishuo",
    "college"  : "Hebei University Of Technology",
    "e-mail"   : "zhangxishuo1998@gmail.com",
    "location" : "TianJin"
}

这就是一个Json对象，我们可以借助菜鸟工具在线解析Json，https://c.runoob.com/front-end/53

object        {4}
    name     : zhangxishuo
    college  : Hebei University Of Technology
    e-mail   : zhangxishuo1998@gmail.com
    location : TianJin

Json数据被解析为一个含有四个属性的对象。

数组

Json数组包含在方括号内，一个数组中可以包含多个对象。

{
    "students" : [
        {
            "name"     : "zhangxishuo",
            "college"  : "Hebei University Of Technology",
            "e-mail"   : "zhangxishuo1998@gmail.com",
            "location" : "TianJin"
        },
        {
            "name"     : "qiuxueying",
            "college"  : "Harbin Normal University",
            "e-mail"   : "350322953@qq.com",
            "location" : "Harbin"
        }
    ]
}

这里students的值为一个数组，数组中包含的是两个对象，对象间用逗号隔开。

总结

了解了Json，我们重新认识一下微信小程序的Json文件。

{
    "pages": [
        "pages/index/index",
        "pages/logs/index"
    ],
    "window": {
        "navigationBarTitleText": "Demo"
    },
    "tabBar": {
        "list": [{
            "pagePath": "pages/index/index",
            "text": "首页"
        },{
            "pagePath": "pages/logs/logs",
            "text": "日志"
        }]
    },
    "networkTimeout": {
        "request": 10000,
        "downloadFile": 10000
    },
    "debug": true
}

对象中有五个参数，分别为pages、window、tabBar、networkTimeout、debug，pages的值为数组，window、tabBar与networkTimeout的值为对象，且tabBar中list的值为一个对象数组，debug的参数为布尔型变量。
